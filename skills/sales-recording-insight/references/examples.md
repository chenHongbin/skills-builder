# 完整示例

## 单通分析示例

**用户输入：** `分析这个录音，小李的，客户是某装修公司`

**执行过程：**

1. 找到录音文件 → 路径确认
2. 转写：Whisper medium 模型，1 分 30 秒完成
3. 分析：逐维度生成 JSON
4. 生成报告：`26.5.20_小李_装修公司_录音洞察.docx`

**返回给用户的摘要：**

```
✅ 录音分析完成！

📄 报告已保存：桌面/录音洞察/小李/26.5.20_小李_装修公司_录音洞察.docx

📊 核心摘要：
客户：某装修公司 | 王经理 | 8分32秒
客户意向：🟢 高（主动问价、要案例、约演示）
综合评分：6.8/10
最强维度：价值表达（8分）——用案例讲清楚了 ROI
最弱维度：推进下一步（5分）——只说"保持联系"，没约具体时间
关键标签：价格异议、跟进机会、需求挖掘充分

💡 管理洞察：
- 核心问题：销售不会处理价格异议，一遇抗就掉节奏
- 辅导优先级：异议处理话术 > 推进下一步意识
- 客户值得重点跟进，有明确需求和预算
- 明天一对一建议练"先认同→再反问→再给方案"模型

🎯 立即改进：
下次遇到客户说贵，换成：
"理解您的顾虑，很多客户一开始也有这个感觉。方便问下您现在的预算是多少吗？我可以帮您看看怎么配置最合适。"
```

## 批量分析示例

**用户输入：** `帮我分析这周团队所有人的录音，大概 20 通`

**执行过程：**

1. 确认录音文件位置 → 发现 20 通录音分布在不同日期
2. 批量转写：逐通 Whisper 转写，预计 15-20 分钟
3. 逐通分析：每通按 6 大维度生成 JSON
4. 团队汇总：生成横向对比和团队诊断
5. 生成报告：`26.5.20_团队_周汇总_录音洞察.docx`

**返回给用户的摘要：**

```
✅ 团队周度分析完成！

📄 报告已保存：桌面/录音洞察/团队/26.5.20_团队_周汇总_录音洞察.docx

📊 团队概览（本周 20 通录音）：
- 平均综合评分：6.2/10（上周 5.8，↑ 0.4）
- 高意向客户：5 通（25%）
- 成交信号：2 通（10%）
- 需重点跟进：8 通（40%）
- 销售失误：3 通（15%）

🏆 本周标杆：
- 评分最高：小王 8.5/10（客户：某教育机构）
- 亮点：开场用"您现在每个月线索成本多少"直击痛点，异议处理用了案例对比法

⚠️ 本周需关注：
- 评分最低：小张 3.5/10（连续 2 通低于 4 分）
- 共性问题：80% 的销售不会在结尾推进下一步，只说"保持联系"
- 新问题：本周出现 3 次"过度承诺"（"保证效果""肯定能成"）

📈 趋势变化：
- 进步最大：小李（上周 5.2 → 本周 6.8，异议处理进步明显）
- 退步最大：小陈（上周 7.1 → 本周 5.3，开场变得敷衍）

🎯 本周训练主题：
"如何优雅地推进下一步动作"——从"保持联系"到"周二下午 3 点我带案例过来"

👀 关注名单：
- 小张：连续 2 通评分低于 4 分，建议明天一对一
- 小刘：有 1 次过度承诺，需提醒红线

💡 明日管理动作：
- 早上：把高意向 5 通分配给对应销售，确认跟进状态
- 中午：15 分钟短会，放小王的开场录音（标杆学习）
- 晚上：一对一辅导小张，练"推进下一步"话术
```

## 管理节奏参考

### 早会前（8:30-9:00）：看数据，定重点

看 AI 生成的昨日/本周分析摘要，重点抓：
- 高意向客户列表（标签含"高意向客户"或"成交信号"）
- 异常通话（评分低于 5 分或标签含"销售失误"）
- 销售问题排行（谁的问题最多）
- 今日重点跟进客户（标签含"跟进机会"且未推进下一步）
- 今日重点训练动作（团队共性问题）

**目标：** 明确今天要抓什么。

### 中午（12:30-13:00）：抓过程，做纠偏

15-30 分钟短复盘：
- 上午销售是否执行到位（有没有按建议改进）
- 有没有新的高意向客户
- 挑一通 5 分钟以下的典型问题录音，团队一起听
- 是否需要立刻调整话术
- 是否需要主管介入重点客户

**目标：** 及时纠偏，不让问题拖到晚上。

### 晚上（17:30-18:00）：做复盘，沉淀打法

看全天/本周数据：
- 今天谁做得好（评分最高的，亮点是什么）
- 今天谁需要辅导（评分最低的，问题是什么）
- 今天出现了哪些客户异议（新的异议类型要更新话术库）
- 今天哪套话术有效（被多个销售使用且效果好的）
- 明天团队要统一强化什么动作

**目标：** 把一天的销售动作变成可沉淀的组织经验。

### 每周五：团队复盘会

- 团队整体通话质量趋势（本周平均分 vs 上周）
- 高转化销售的共同特征（他们做了什么不一样）
- 低转化销售的主要问题（集中在哪个维度）
- 客户异议变化趋势（这周客户最关心什么）
- 产品表达是否需要调整（客户对价值的理解有没有偏差）
- 销售 SOP 是否需要更新

**目标：** 让销售团队持续进化。
